# kotlin-cli-gradle-spring-jacoco-cucumber-junit-hello-world

## Description
A POC for spring app using junit,
jacoco, and cucumber framework.

This is a simple and trivial way to start:
  - kotlin
    - springframework
    - cucumber test runner for junit5
  - cucumber
    - parameterized scenario

## Tech stack
- kotlin
- gradle
  - spring
  - junit
  - jacoco
  - cucumber

## Docker stack
- gradle:jdk11

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
